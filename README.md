# 💙 Voll.med API

Projeto desenvolvido durante a formação de Spring Boot da Alura 🚀

A aplicação simula o backend de uma clínica médica fictícia chamada **Voll.med**, responsável pelo gerenciamento de médicos, pacientes e consultas através de uma API RESTful desenvolvida com Java e Spring Boot.

O foco do projeto foi praticar desde os fundamentos de APIs REST até conceitos mais avançados de arquitetura, segurança, boas práticas e testes automatizados.

---

# 💻 Sobre o projeto

A Voll.med é uma clínica médica fictícia que precisa de um sistema para gerenciamento de consultas médicas.

A API desenvolvida permite:

- cadastro de médicos
- cadastro de pacientes
- agendamento de consultas
- cancelamento de consultas

Enquanto um time hipotético seria responsável pelo aplicativo mobile, este projeto representa o desenvolvimento completo da API REST da aplicação 🩺⚙️
